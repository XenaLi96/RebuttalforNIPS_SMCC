Thus, genuinely new primary acquisition represents \(4/70=5.7\%\) of unique
source samples, \(9{,}806/23{,}953{,}632=0.041\%\) of measured/source units, and

the resource's scale comes predominantly from public-data aggregation and
harmonization, while the genuinely new primary component is small.

\questionsep
\textbf{7Gar-5.}\quad
\revcomment{STBoost, STBoost-Ref, ``Ours,'' STBoosted BLEEP, BLEEP, Table 2, and the
inference protocol are unclear.}

\response
We will use one consistent terminology:

\begin{itemize}
  \item \textbf{STBoost} is the data-interface framework that converts a compatible
  spot-level pipeline to cell-aligned training and evaluation records.
  \item \textbf{BLEEP} is the published image-to-expression baseline.
  \item \textbf{STBoosted BLEEP} is BLEEP retrained and evaluated through the
  cell-aligned STBoost interface.
  \item \textbf{STBoost-Ref} is our native retrieval-based reference predictor;
  ``Ours'' in Figure 4 refers to STBoost-Ref and will be renamed accordingly.
\end{itemize}

At query time, STBoost-Ref receives only the local and wider-context histology crops
and outputs a cell-aligned expression vector. Query expression is never an input.
Expression is used only to build training-split reference entries. Table 2 will be
retitled to define a ``cell unit'' as one aligned cell record and will point forward
to the construction and split sections. We will introduce BLEEP and STBoost before
their first use, define AK/AD/AL in the text, add the missing task/metric/ratio
definitions to Figure 2, and remove repetitive framing.
